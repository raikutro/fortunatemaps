Hey! I made some changes.

  • Created a dedicated Browse page to search and filter maps.
  • Added the ability to "Like" maps.
  • Implemented a custom-trained perceptual hash algorithm to process and map visuals directly into a 256-bit representation for a future map similarity database.
  • Added Date filtering (Before Date / After Date) options to the Browse maps page.
  • Permanently disabled commenting on maps older than 7 days (1 week) to ensure discussions remain isolated to active revisions.
  • Added a Dark Mode override. It can be toggled via the new checkbox in the User Settings page.
